## 📋 About DIGITOOLS

DIGITOOLS is a powerful e-commerce platform designed to showcase and sell premium digital products and services. From AI writing tools to design templates and stock assets, DIGITOOLS provides a seamless shopping experience with an intuitive interface and robust functionality.

---

## ✨ Key Features

### 🛒 **Smart Shopping Cart System**
Add, remove, and manage products with ease. Real-time cart updates with instant notifications using React-Toastify to keep users informed about every action.

### 💎 **Beautiful UI with DaisyUI**
Premium user interface components powered by DaisyUI and Tailwind CSS. Responsive design that works flawlessly on mobile, tablet, and desktop devices with stunning visual aesthetics.

### 📦 **Dynamic Product Catalog**
Browse through a curated collection of digital tools including AI writing solutions, design templates, stock assets, and automation tools. JSON-based product data management for easy updates and scalability.

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
```bash
git clone <repository-url>
cd DIGITOOLS
```

2. **Install dependencies**
```bash
npm install
```

3. **Run the development server**
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm preview
```
